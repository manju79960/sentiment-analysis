# 🤖 Sentiment Analysis Web App

This project is a **real-time Sentiment and Emotion Analyzer** built using **Python, NLP, and Streamlit**.  
The application analyzes user input text and predicts the sentiment as **Positive, Neutral, or Negative** and detects basic emotions.

---

## 🚀 Features

- Real-time sentiment analysis
- Emotion detection from text
- Interactive Streamlit web interface
- Sentiment strength indicator
- Clean and simple UI

---

## 🛠 Technologies Used

- Python
- Streamlit
- NLTK (VADER Sentiment Analyzer)
- Natural Language Processing (NLP)
